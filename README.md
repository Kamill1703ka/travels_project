# 🌍 Travel Telegram Bot ("Давай уедем")

## 📌 Описание проекта
**Travel Telegram Bot** — это Telegram-бот, который помогает пользователям получать информацию для путешествий.  
Бот умеет:

- показывать информацию о странах
- показывать погоду в городах
- отвечать на вопросы о путешествиях с помощью ИИ
- анализировать тему сообщения
- определять тональность текста

Бот реализован на языке **Python** с использованием Telegram API, внешних REST API и моделей машинного обучения.

Главные команды бота:

- /start - запуск работы бота
- /help - помощь
- /country <страна>
- /weather <город>
- /ask <вопрос>
- /exit - выход из бота

## 2️⃣ REST Countries API
API используется для получения информации о странах.

Источник:  
https://restcountries.com/

Функциональность:

- название страны
- столица
- население
- языки

# 🔌 Используемые API

## 1️⃣ OpenWeather API
API используется для получения информации о погоде.

Источник:  
https://openweathermap.org/api

Функциональность:
- получение температуры
- описание погоды
- влажность
- скорость ветра

# 🤖 Базовая модель искусственного интеллекта

Для генерации ответов используется модель:

**DeepSeek Chat**

Доступ осуществляется через платформу **OpenRouter API**.

Модель используется для:

- генерации туристических рекомендаций
- ответов на вопросы пользователя
- составления списков достопримечательностей
- выдачи советов путешественникам
